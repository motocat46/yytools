# pkg — 核心功能模块

## 模块一览

| 模块 | 简介 |
|------|------|
| [algorithms](algorithms/README.md) | 排序、二分查找、数学工具、唯一 ID 生成 |
| [ds](ds/README.md) | 堆、队列、栈、有序集合 |
| [mechanics](mechanics/README.md) | 游戏规则机制：渐进式权重周期、双层保底抽卡引擎 |
| [infra](infra/README.md) | panic 安全执行、时间工具、OS 封装、无界 Channel |
| [common](common/README.md) | 泛型类型约束、运行时断言框架 |
| [slicex](slicex/README.md) | 切片工具函数（MinBy、MaxBy、MinInSlice 等） |
| [numconst](numconst/README.md) | 常用数字常量（千/万/亿）和时间单位常量 |
