# pkg/algorithms

算法实现集合，包含排序、数学工具、二分查找、唯一 ID 生成等。

| 子模块 | 功能 |
|--------|------|
| [sort](sort/README.md) | 多种排序算法（快排、计数排序、基数排序等） |
| [binary_search](binary_search/README.md) | 二分查找（精确查找、左右边界查找） |
| [mathx](mathx/README.md) | GCD、Abs、Min/Max 及数学子模块索引 |
| [idgen/snowflake](idgen/snowflake/README.md) | 无锁雪花算法唯一 ID 生成器 |
| [mathx/sampling](mathx/sampling/README.md) | Floyd 算法不重复采样、带最小间隔约束采样 |
