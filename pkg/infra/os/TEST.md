# os 测试说明

## 测试文件

| 文件 | 覆盖范围 |
|------|---------|
| `os_test.go` | `IsFileExist`：存在文件/不存在文件/空路径/目录；`IsFileNormalStat`：同上四类；`BackupFile`：备份存在/不存在/空路径的文件，多次备份同一文件（时间戳唯一性），含空格的特殊文件名，100 次连续备份 |

## 分层执行命令

```bash
# 快速验证
go test ./pkg/infra/os/

# 竞态检测
go test -race ./pkg/infra/os/
```

## 注意

- 测试使用 `t.TempDir()` 创建临时目录，测试结束后自动清理，无副作用
- `BackupFile` 备份文件名格式：`<原文件名>_<时间戳>`，多次备份依赖时间戳区分，测试中加了 10ms 延迟确保时间戳不同
- 无基准测试——文件系统 I/O 性能受环境影响大，不在此测量
